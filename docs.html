BLOOD Documentation

AI Liquidation Tracker - Get your information exactly where you need it most

Overview

Blood is an advanced AI-powered liquidation tracker that monitors the largest liquidations across the highest volume cryptocurrency markets in real-time. Get critical market intelligence exactly where you need it most for optimal trading decisions.

MARKET INTELLIGENCE

Blood tracks liquidations across major exchanges, providing instant alerts and analytics to help you capitalize on market movements before the crowd catches on.

Key Features

  • Real-time liquidation monitoring across 50+ major exchanges
  • AI-powered pattern recognition for liquidation cascades
  • Instant notifications for large liquidation events
  • Historical liquidation data and trend analysis
  • Integration APIs for trading bots and algorithms
  • Auto-buyer functionality for strategic positioning

Supported Markets

Blood monitors liquidations across the highest volume markets:

  • Perpetual Futures: BTC, ETH, SOL, BNB, XRP, ADA, DOGE
  • Options: Major crypto options across Deribit and OKX
  • Spot Margin: High-volume spot margin positions
  • Cross-Exchange: Multi-exchange liquidation correlation tracking

Integration

Quick Setup

# Install Blood SDK
npm install @blood/liquidation-tracker

# Initialize with API key
const Blood = require('@blood/liquidation-tracker');
const tracker = new Blood({
  apiKey: 'your-api-key-here',
  environment: 'production'
});

# Start monitoring
tracker.start();

WebSocket Integration

Connect to Blood's real-time liquidation feed for instant market intelligence: 

# Connect to WebSocket feed
const ws = new WebSocket('wss://api.blood.io/liquidations');

ws.on('message', (data) => {
  const liquidation = JSON.parse(data);
  console.log(`Large liquidation detected: ${liquidation.amount} ${liquidation.symbol}`);
  
  // Process liquidation event
  if (liquidation.amount > 1000000) {
    handleLargeLiquidation(liquidation);
  }
});

# Subscribe to specific markets
ws.send(JSON.stringify({
  action: 'subscribe',
  markets: ['BTC-PERP', 'ETH-PERP', 'SOL-PERP']
}));

REST API Integration

Access historical liquidation data and analytics through our REST API: 

# Get recent large liquidations
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.blood.io/v1/liquidations?size=large&timeframe=1h"

# Get liquidation heatmap data
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.blood.io/v1/heatmap?market=BTC-PERP&interval=15m"

# Get liquidation cascade predictions
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.blood.io/v1/predictions/cascades"

Trading Bots

Bot Integration

Integrate Blood's liquidation intelligence directly into your trading bots for enhanced performance and strategic positioning: 

# Trading bot configuration with Blood integration
const BloodBot = require('@blood/trading-bot');

const bot = new BloodBot({
  apiKey: 'your-blood-api-key',
  exchange: 'binance', // or 'bybit', 'okx', etc.
  strategy: 'liquidation-hunter',
  riskManagement: {
    maxPositionSize: 10000,
    stopLoss: 0.02,
    takeProfit: 0.05
  }
});

# Configure liquidation-based triggers
bot.on('largeLiquidation', (event) => {
  if (event.amount > 5000000 && event.market === 'BTC-PERP') {
    bot.execute('contrarian-entry', {
      size: calculatePositionSize(event),
      direction: opposite(event.side)
    });
  }
});

bot.start();

Configuration Options

Configure the auto buyer to match your investment strategy and risk tolerance:

  • Budget Allocation: Set daily, weekly, or monthly purchase limits
  • Trigger Conditions: Buy on large liquidations, price dips, or time intervals
  • Slippage Tolerance: Maximum acceptable slippage for trades (0.1% - 5%)
  • Gas Optimization: Smart gas pricing to minimize transaction costs

Setup Instructions

# Configure Blood auto buyer
const autoBuyer = new BloodAutoBuyer({
  wallet: 'your-wallet-address',
  privateKey: process.env.PRIVATE_KEY,
  budget: {
    daily: 1000, // $1000 daily limit
    perTrade: 100 // Max $100 per trade
  },
  triggers: {
    largeLiquidation: true, // Buy on $5M+ liquidations
    priceDrops: 0.05, // Buy on 5%+ drops
    timeInterval: '4h' // Buy every 4 hours
  },
  slippage: 0.005, // 0.5% max slippage
  gasStrategy: 'optimal'
});

autoBuyer.start();
